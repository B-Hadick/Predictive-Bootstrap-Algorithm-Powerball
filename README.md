run and stick python programs for running a predictive analysis of all available historical
through current lottery numbers for the PowerBall United States lottery
and bootstrapping before providing the top 5 most likely sets of numbers to be drawn in decreasing probability, with likelihood percentages.
--------------------------------------------------------------------------------------------------------------------------------------------
--Instructions--
1) run 'run_powerball_py.bat' as admin.
2) run 'powerball_predict.py' as admin
3) wait ~30 seconds.
4) enjoy results
--------------------------------------------------------------------------------------------------------------------------------------------
--ADVISE--
due to the nature of random selection lottery, no true predictive model is possible. This is simply an algorithm to analyze and select the 
MOST LIKELY numbers to be randomly selected based on individual frequency, paired frequency, cumulative pattern analysis, time-passed 
grading degredation (length of time passed vs last drawn, initial relevancy score of 100).
